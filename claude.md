# 项目指令：神经网络对抗攻击与防御研究

## 1. 项目概况

本项目旨在研究深度学习模型的安全性，主要针对 ResNet50 模型进行对抗样本攻击（FGSM/PGD）与防御实验。

- **环境**: Python, PyTorch, NVIDIA RTX 5060 GPU。
- **操作系统**: Windows
- **目标**: 实现从模型部署、攻击实施到防御加固的完整流程，并记录实验日志。
- **计算设备**: NVIDIA GeForce RTX 5060 Laptop GPU
- **编程语言**: Python 3.13
- **深度学习框架**: PyTorch (Preview / Nightly) + CUDA 13.0
- **Torchvision**: 0.21.0+cu124
- **Streamlit**：version 1.57.0

## 2. 核心工作流规范

每次执行任务时，请遵循以下流程：

1. **代码实现/修改**: 编写符合逻辑的 Python 代码。在每次新建一个文件的时候在文件开头使用注释的方式介绍这个代码文件是干什么的
2. **文档记录**: 在 `EXPERIMENT_LOG.md` 中自动追加本次操作的记录（包含时间、修改内容、实验预期）在NOTE.md中自动追加本次操作的最核心原理，比如梯度下降算法的原始公式和论文地址，不要过度冗长。
3. **GitHub 提交指令**: 完成代码修改后，主动提供一条 `git commit` 指令建议，格式为 `feat: [简述]` 或 `fix: [简述]`。
4. **报告书写**：完成一个较大的阶段之后完善report.md中的实验报告，要求包含环境，实验原理，实验代码，不要每次更改代码都书写报告

## 3. 技术约束与规范

- **模型处理**: 
  - 使用预训练的 ResNet50。
  - 演示准度时，需输出 Top-1 和 Top-5 置信度。
- **攻击算法**: 
  - 核心使用基于梯度的算法（如 FGSM, PGD）。
  - 攻击演示需包含：原始图像及其预测、扰动后的对抗样本及其错误预测（例如：将“狗”识别为“鸡”）。
- **防御手段**: 
  - 探索包括但不限于：对抗训练（Adversarial Training）、图像预处理（高斯模糊/压缩）等。
- **代码风格**: 
  - 使用面向对象编程，保持代码整洁。
  - 关键步骤（如计算梯度）必须有中文注释。

## 4. 阶段性任务清单

请按顺序引导我完成以下任务：

### 阶段一：部署与验证

- 编写脚本加载 ResNet50 模型。
- 遍历 `testset` 文件夹，输出图像名称、分类结果和置信度。
- 确保充分利用 RTX 5060 的 CUDA 加速。

### 阶段二：对抗攻击实施

- 实现梯度下降算法（FGSM/PGD）生成扰动。
- 演示攻击成功案例：记录原始标签与攻击后的错误标签。
- 可视化扰动部分（Noise Visualizing）。

### 阶段三：防御与加固

- 实现至少一种防御手段。
- 对比防御前后模型对相同对抗样本的识别率。

### 阶段四：项目总结

- 汇总所有实验数据，生成最终报告。
